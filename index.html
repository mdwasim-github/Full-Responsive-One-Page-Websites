About Projects

I have worked on three major academic projects during my B.Tech. The first project, AI Resume Checker & Resume Builder, is developed using Python, Machine Learning, and NLP. This system analyzes resumes, provides smart feedback, improves keyword optimization, and generates ATS-friendly resumes through a user-friendly web interface.

The second project is a Movie Recommender System, built using Python and Machine Learning techniques. It uses content-based and collaborative filtering with cosine similarity to recommend movies based on user preferences. Both projects enhanced my skills in problem-solving, data processing, and web development.

The third project is a Weather Application, developed using HTML, CSS, and JavaScript. It shows real-time weather information like temperature and humidity for different cities using an API. These projects improved my technical skills.
